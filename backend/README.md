# Backend do Sistema de Chamados

Arquivo de automacao da base de dados e evidencias.

## Preparacao

1. Crie uma planilha vazia para os dados.
2. Abra o editor de automacao vinculado a essa planilha.
3. Cole o conteudo de `Code.gs`.
4. Ajuste `CONFIG`.
5. Execute `setupManual` uma vez.
6. Publique como aplicativo web com permissao de acesso por link.
